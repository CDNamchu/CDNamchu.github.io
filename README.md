## ğŸš€ Getting Started

This blog is built with Jekyll and hosted on GitHub Pages.

### Local Development

```bash
# Clone the repository
git clone https://github.com/CDNamchu/CDNamchu.github.io.git
cd CDNamchu.github.io

# Install Jekyll and dependencies
bundle install

# Run local development server
bundle exec jekyll serve

# Visit http://localhost:4000 in your browser
```

## ğŸ“ Writing Content

### Creating a New Post

1. Navigate to the `_posts` directory
2. Create a new Markdown file: `YYYY-MM-DD-post-title.md`
3. Add frontmatter metadata:

```yaml
---
title: "Your Post Title"
date: 2025-11-20
categories: [Category1, Category2]
tags: [tag1, tag2, tag3]
---
```

4. Write your content using Markdown
5. Commit and push - GitHub Pages will automatically build and deploy
